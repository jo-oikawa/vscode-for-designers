# Examples

Runnable example projects for designers building in VS Code with Copilot.

Each example is self-contained — you can open it in VS Code, follow the instructions inside, and have something working quickly.

---

## Index

| Example | Description |
|---------|-------------|
| [`task-dashboard/`](task-dashboard/) | Turn simple markdown files into a visual task board and notepad — viewable inside VS Code |

*Want to contribute an example? See the [contributing guide](../README.md#contributing).*

---

## How to use an example

1. Open the example folder in VS Code (`File > Open Folder`)
2. Read the `README.md` inside the example for setup instructions
3. Open GitHub Copilot Chat and start exploring
